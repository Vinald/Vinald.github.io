## Troubleshooting 🔧

### **If form doesn't send:**
1. Check browser console (F12)
2. Verify all 3 IDs are correct in script.js
3. Make sure your email service is connected in EmailJS
4. Check that the template exists

### **If you don't get email:**
1. Check spam/junk folder
2. Verify email service is properly connected
3. Test with a test email in EmailJS dashboard
4. Make sure you're using the correct email address

### **If button keeps loading:**
1. Check if there's an error in browser console
2. Verify internet connection
3. Try refreshing the page
